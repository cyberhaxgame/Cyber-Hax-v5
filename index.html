Quick Rules

1. Reach the server node before the other player.

2. Use `probe`, `sweep`, and `path` to read the graph.

3. Locked nodes need keys or passwords. Mines and traps can stun you.

4. After a match, use rematch to keep score or restart to clear the room score.
